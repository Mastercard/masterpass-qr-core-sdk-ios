AbstractData

@interface AbstractData : NSObject {
  NSMutableDictionary<TagInfo *, id> *map;
}

Abstract class AbstractDataModel is the root of Data hierarchy, containing the basic common functions applicable to all Data classes. All Data classes will have AbstractDataModel as its superclass.
