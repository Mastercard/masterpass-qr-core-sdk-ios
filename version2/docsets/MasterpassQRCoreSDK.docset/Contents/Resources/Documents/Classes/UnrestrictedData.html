UnrestrictedData

@interface UnrestrictedData : TemplateData

This class is to store Unreserved Template Data of Push Payment Data tags 80-99. This class is to store Unreserved Template Data of Additional Data tags 50-99.

Note

To assign Application Identifier (tag 00), use AID property from TemplateData class.
